# S.Job — превью сайта

Автономные (self-contained) HTML-файлы для демонстрации. Каждый работает
офлайн — все стили, шрифты и скрипты встроены внутрь.

| Файл | Что |
|---|---|
| `index.html` | Главная-хаб: выбор между тремя вариантами превью. |
| `web.html` | Веб-сайт: лендинг + лента вакансий + страница вакансии. |
| `mobile.html` | Мобильное приложение: онбординг, лента, детальная вакансия. |
| `slides.html` | Презентация проекта (9 слайдов, стрелки ← / →). |

## Как открыть
Просто откройте `index.html` в браузере — и переходите по карточкам.

## GitHub Pages
1. Запушьте содержимое этой папки в репозиторий.
2. Settings → Pages → Source: ветка `main`, папка `/ (root)` (или `/docs`,
   если положите файлы туда).
3. Сайт будет доступен по адресу `https://<user>.github.io/<repo>/`.
